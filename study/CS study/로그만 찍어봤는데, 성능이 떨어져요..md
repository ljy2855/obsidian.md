### 문제 상황

```python
def solution(n, stations, w):
    answer = 0
    start = 1
    
    for station in stations:
        count = station - w - start
        if count > 0:
            answer += math.ceil(count / (2 * w + 1))
        start = station + w + 1
    
    if start <= n:
        count = n - start + 1
        answer += math.ceil(count / (2 * w + 1))
    
    return answer
```
